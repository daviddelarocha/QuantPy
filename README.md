# Descripción del Proyecto

## Herramientas Cuantitativas para Inversión con Python

Este repositorio reúne scripts, notebooks y módulos en Python diseñados para analizar datos financieros, evaluar acciones y construir estrategias cuantitativas que apoyen decisiones de inversión más informadas.  
El objetivo es ofrecer herramientas transparentes, educativas y reproducibles, ideales para quienes buscan aprender análisis cuantitativo y fundamentos de inversión.

Estas herramientas acompañan el contenido del canal de YouTube asociado, donde se explica paso a paso cada concepto financiero y su implementación en Python.

---

## ¿Qué incluye este repositorio?

### 1. Obtención y limpieza de datos
- Descarga de precios históricos (OHLCV)  
- Preprocesamiento y normalización  
- Ajustes por splits y dividendos  

### 2. Análisis exploratorio de datos (EDA)
- Estadísticas de retorno  
- Volatilidad  
- Correlaciones  
- Visualizaciones avanzadas de series de tiempo  

### 3. Indicadores y análisis técnico
- SMA, EMA, RSI, MACD, Bollinger Bands  
- Generación de señales simples de compra/venta  

### 4. Análisis fundamental automatizado
- Ratios financieros  
- Evaluación básica de calidad empresarial  
- Construcción de un "score fundamental" configurable  

### 5. Gestión de riesgo y portafolios
- Sharpe Ratio  
- Drawdown  
- Optimización de portafolios (Markowitz)  

### 6. Backtesting
- Motor básico de backtesting  
- Reportes y métricas de rendimiento  
- Comparación con benchmarks  

### 7. Apps y automatización
- Dashboards interactivos con Streamlit  
- Scripts automatizados de análisis  

---

## Objetivo del proyecto

Crear un marco accesible para aprender y enseñar análisis cuantitativo en Python, promoviendo prácticas responsables orientadas al entendimiento del riesgo y a la toma de decisiones informada.  
Este proyecto es exclusivamente educativo y no representa asesoría financiera personalizada.

---

## Tecnologías principales
- Python 3.x  
- Pandas, NumPy  
- Matplotlib / Plotly  
- yfinance  
- Scikit-Learn  
- Streamlit o Dash (opcional)
